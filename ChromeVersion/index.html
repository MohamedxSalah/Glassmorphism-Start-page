How to get your iCal URL
Google Calendar: Open calendar settings → "Integrate calendar" → copy the "Public address in iCal format" link.

Outlook: Settings → View all → Calendar → Shared calendars → Publish → copy ICS link.

Apple Calendar: Right-click calendar → Share Calendar → enable "Public Calendar" → copy link.
